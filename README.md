xe 유튜브 게시판 스킨
====================

미리보기 <http://instantpage.kr/home/board_Dghz69>


# 사용법
스킨설치후 해당 스킨으로 변경(INSP 유튜브) 그리고 게시판 리스트에 접속 접속시 필요한 사용자정의(extra) 알아서 설정됨.

# 기능
- 유튜브 영상 주소 붙여서 사용
- 리스트에 유튜브 썸네일을 자동으로 불러옵니다. 이미지를 별도로 저장하지 않습니다. 트래픽은 유튜브가.
- 그외에 그냥 기존 게시판 기능들.
- 옵션 (자동재생, 재생후 추천 영상 옵션, 시간설정)
